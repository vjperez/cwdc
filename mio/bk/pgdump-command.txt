To rewrite data, you should either drop tables first or pg_dump -F c and pg_restore -c.
Warning - in both cases it will destroy old data (this seems what you want though)


pg_dump.exe -U victordbu -d nepeype -t dueno -t nepe -t bregando -t '*id_seq' -f C:\Users\victor\Desktop\nepeype.sql

pg_dump.exe -U victordbu -d 4bolas -t foto  -t entrada -t '*id_seq' -f C:\Users\victor\Desktop\4bolas.sql
